# Multi-Disease Prediction System Using Machine Learning

## Overview

This project presents a machine learning-based disease prediction system that classifies patient records into **Diabetes** or **Heart Disease**. The datasets were preprocessed, transformed into a common feature structure, and merged into a single dataset for classification.

## Datasets

- Diabetes Dataset
- Heart Disease Dataset

Disease Labels:
- 0 → Diabetes
- 1 → Heart Disease

## Machine Learning Models

- Logistic Regression
- Support Vector Machine (SVM)
- Random Forest
- XGBoost

## Results

| Model | Accuracy |
|--------|----------|
| Logistic Regression | 98.60% |
| SVM | 99.53% |
| Random Forest | 100.00% |
| XGBoost | 99.53% |

## Workflow

- Data Loading
- Data Preprocessing
- Dataset Merging
- Feature Selection
- Train-Test Split
- Feature Scaling
- Model Training
- Performance Comparison

## Conclusion

The project demonstrates the use of supervised machine learning algorithms for disease classification. Among all models, Random Forest achieved the highest accuracy.
